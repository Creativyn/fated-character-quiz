# Personality Quiz Web App

A fully data-driven, accessible personality quiz built with pure HTML, CSS, and JavaScript.  
No frameworks. No build tools. Just drop it into a folder and run.
Don't use this app without receiving my permission first.

---

## Features

- 15-question personality quiz
- 10 personality outcomes
- Data-driven architecture (edit config & questions without touching logic)
- Fully accessible (WCAG 2.2 AA–friendly)
- One-page layout (all questions visible)
- Required answer validation
- Dynamic scoring system
- Results sorted from highest to lowest match
- Percentage breakdown for all personalities
- Personalized result cards with images and themes
- Dark mode support (system-based)
- Smooth scrolling (respects reduced motion settings)
- Print-friendly results layout
- Retake Quiz button
- Share Results button (Web Share API + clipboard fallback)
- Optional URL sharing support (query-string based)
